## parameter
### generate new parameter
run the parameter.py using script or module mode
```sh
python parameter.py
```
or
```sh
python -m parameter
```

## sym visulization and check
run the parameter.py using script or module mode
```sh
python sym.py
```
or
```sh
python -m sym
```

## train and eval
use module mode to run, not script
### train
```sh
python -m  module_name.train
```
### eval
```sh
python -m  modulename.eval
```
